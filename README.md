## 📖 项目简介

**壹零贰肆少儿编程机构管理系统** 是一个专为少儿编程培训机构设计的综合管理平台，提供学生档案管理、课程管理、财务管理、签到考勤、数据分析等核心功能。系统采用纯前端技术栈，数据存储在浏览器本地，无需后端服务器，部署简单，使用便捷。

### 🎯 适用场景

- 少儿编程培训机构
- 中小型教育机构
- 兴趣班、辅导班管理
- 课外培训机构

## ✨ 功能特性

### 📊 系统主页
- **数据统计仪表盘**：实时显示学生总数、课程数量、今日签到、本月收入等关键指标
- **课程表展示**：可视化周课程安排，支持导出为PNG图片
- **快捷导航**：快速访问各功能模块
- **数据备份与恢复**：一键导出/导入所有系统数据

### 👨‍🎓 学生管理
- **学生档案管理**
  - 基本信息：姓名、年龄、性别、联系方式、家长信息
  - 课程信息：所选课程、剩余课时（支持0.5课时精度）
  - 学习状态：积分、星级评定
  - 学生照片：支持上传和预览
- **高级搜索与筛选**
  - 按姓名、课程、星级快速搜索
  - 多条件组合筛选
- **消课记录管理**
  - 记录每次课程消耗
  - 自动计算剩余课时
  - 支持批量导出
- **积分系统**
  - 积分增减记录
  - 历史记录查询
  - 积分排行榜
- **数据导出**
  - Excel格式导出学生信息
  - 包含完整的学习记录

### 📚 课程管理
- **课程信息维护**
  - 课程名称、级别、描述
  - 课程容量：统一设置为20人
  - 已报名人数自动统计
- **内置课程体系**
  - Scratch编程（初级/中级/高级）
  - Python编程（初级/中级/高级）
  - C++编程（初级/中级）
  - 机器人编程（初级/中级）
  - 校内辅导课
- **课程数据分析**
  - 报名情况统计
  - 容量使用率分析

### 💰 财务管理
- **收支记录**
  - 收入记录：学费、报名费等
  - 支出记录：运营成本、采购费用
  - 详细备注信息
- **财务统计**
  - 总收入、总支出、净利润
  - 按时间段筛选
  - 月度/年度报表
- **数据导出**
  - Excel格式财务报表
  - 支持筛选导出

### ✅ 签到管理
- **每日签到**
  - 学生签到/签退
  - 手动输入时间
  - 状态选择：正常、迟到
- **考勤统计**
  - 今日出勤人数
  - 本周/本月出勤率
  - 迟到、缺勤统计
- **请假管理**
  - 请假申请提交
  - 请假日期范围设置
  - 请假原因记录
  - 请假记录查询与取消
- **历史记录**
  - 按日期范围查询
  - 签到详细信息
  - 出勤状态统计
- **数据导出**
  - 详细签到记录
  - 学生出勤统计
  - 请假记录汇总
  - **周出勤率统计**（按周汇总分析）
  - 支持选择保存路径

### 📈 数据分析
- **学生数据分析**
  - 年龄分布图表
  - 课程选择统计
  - 性别比例分析
- **课程数据分析**
  - 课程报名人数对比
  - 课程容量使用情况
  - 热门课程排行
- **财务数据分析**
  - 收支趋势图
  - 月度收入对比
  - 利润率分析
- **可视化图表**
  - 基于 Chart.js 的数据可视化
  - 柱状图、饼图、折线图
  - 交互式图表展示

**部署优势：**
- ✅ 完全免费

## 📚 使用文档

### 首次使用

1. **访问系统主页**
   - 查看数据统计仪表盘
   - 了解系统功能模块

2. **添加课程**
   - 进入"课程管理"
   - 系统已内置常用课程
   - 可添加自定义课程

3. **添加学生**
   - 进入"学生管理"
   - 点击"添加学生"
   - 填写学生基本信息

4. **开始使用**
   - 记录学生签到
   - 管理消课记录
   - 查看数据分析

### 数据管理

#### 数据存储
- 所有数据存储在浏览器 `localStorage` 中
- 数据仅存储在本地，不会上传到服务器
- 清除浏览器缓存会丢失数据

#### 数据备份
**强烈建议定期备份数据！**

1. **导出备份**
   - 点击主页"导出所有数据"按钮
   - 选择保存位置
   - 自动生成3个文件：
     - JSON 完整备份文件
     - Excel 数据表格
     - PNG 课程表图片

2. **导入恢复**
   - 点击主页"导入数据"按钮
   - 选择之前导出的 JSON 文件
   - 确认导入即可恢复

#### 选择保存路径功能
**支持的浏览器**：Chrome 86+、Edge 86+、Opera 72+

**如何使用**：
- 点击导出按钮后会弹出系统文件选择对话框
- 可选择任意保存位置（桌面、文档、U盘等）
- 可自定义文件名
- 完全像桌面软件一样的体验

### 功能使用指南

#### 学生管理

**添加学生**
1. 点击"添加学生"按钮
2. 填写必填信息：姓名、年龄、性别、联系方式
3. 选择课程、设置剩余课时
4. 可选：上传学生照片
5. 保存

**编辑学生**
1. 在学生列表中点击"编辑"按钮
2. 修改信息
3. 保存更改

**消课记录**
1. 点击学生的"消课"按钮
2. 输入本次消耗的课时（支持0.5课时）
3. 添加备注（可选）
4. 系统自动更新剩余课时

**积分管理**
1. 点击"积分"按钮
2. 增加或减少积分
3. 填写原因
4. 系统记录积分变更历史

#### 签到管理

**每日签到**
1. 进入"签到管理"页面
2. 查看今日学生列表
3. 点击"签到"按钮
4. 选择时间和状态（正常/迟到）

**请假管理**
1. 点击学生的"请假"按钮
2. 选择请假日期范围
3. 填写请假原因
4. 提交请假申请

**取消签到/请假**
- 点击"取消"按钮即可撤销今日签到或请假记录

**查看历史记录**
- 选择日期范围（今日/本周/本月/自定义）
- 查看历史签到详情

**导出考勤数据**
1. 点击"导出签到记录"
2. 选择保存位置
3. 获得包含4个工作表的Excel文件：
   - 详细记录
   - 学生统计
   - 请假记录
   - 周出勤率统计

#### 财务管理

**添加收支记录**
1. 选择类型（收入/支出）
2. 选择分类
3. 输入金额
4. 填写备注
5. 保存

**查看统计**
- 自动计算总收入、总支出、净利润
- 按时间段筛选记录

#### 数据分析

**查看可视化报表**
- 学生年龄分布
- 课程报名统计
- 财务趋势分析
- 交互式图表

## 🛠️ 技术栈

### 前端技术
- **HTML5** - 语义化标签，结构清晰

## ⚠️ 注意事项

### 数据存储
1. **定期备份**：强烈建议定期使用"导出所有数据"功能备份数据
2. **浏览器缓存**：清除浏览器缓存/数据会导致数据丢失
3. **隐私模式**：不建议在浏览器隐私/无痕模式下使用
4. **跨浏览器**：不同浏览器之间数据不共享

### 使用限制
1. **存储容量**：localStorage 通常限制为 5-10MB
2. **并发使用**：不支持多人同时编辑
3. **数据同步**：不支持跨设备数据同步

### 最佳实践
1. **定期备份**：建议每周导出备份一次
2. **数据验证**：导入数据前请确认文件完整性
3. **浏览器更新**：保持浏览器为最新版本
4. **文件管理**：妥善保管备份文件
